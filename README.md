# vue-nwjs

## Project setup
```
npm install
```

### Compiles and hot-reloads web app for development
```
npm run serve
```

### Compiles and minifies web app for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Compiles and hot-reloads nw app for development
```
npm run serve
npm run nw-serve
```

### Build nw app for production
```
npm run nw-serve
```
